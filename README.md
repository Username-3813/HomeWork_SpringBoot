# Электронный гараж 🚗

Веб-приложение для учёта личных транспортных средств, расходов, ремонтов, страховок и документов. Помогает контролировать состояние техники, планировать обслуживание и хранить всю информацию в одном месте.

## 📌 Основные возможности

- **Управление ТС** – добавление, редактирование, удаление автомобилей, мотоциклов, снегоходов и другой техники. Поддерживаются фото, марка, модель, VIN, госномер, пробег/моточасы.
- **История расходов** – все траты по категориям (ремонт, топливо, страховка, штрафы, мойка, прочее). Фильтр по дате, категории и типу ТС, с пагинацией и суммарной статистикой.
- **Ремонты** – запись ремонтов с пробегом, описанием, стоимостью и названием мастерской. Автоматически добавляются в расходы.
- **Страховки** – хранение полисов ОСАГО, КАСКО и других с датами действия и загрузкой скан-копий (лицевая/обратная стороны). Автоматическое напоминание об истечении.
- **Документы** – загрузка любых файлов (ПТС, сервисная книжка, договоры) с привязкой к ТС.
- **Напоминания** – планирование обслуживания с установкой даты и стоимости. Можно отмечать выполненные.
- **Администрирование** – управление пользователями, изменение ролей, просмотр ТС и расходов любого пользователя.
- **Тёмная тема** – переключение светлой/тёмной темы с сохранением выбора в браузере.
- **Уведомления** – иконка с количеством предстоящих напоминаний и истекающих страховок (в шапке сайта).

## 🛠️ Стек технологий

- **Java 17+**
- **Spring Boot 3.x**
- **Spring Data JPA** (Hibernate)
- **Spring Security**
- **MariaDB / MySQL** (база данных)
- **Thymeleaf** (шаблонизатор)
- **Bootstrap 5** + **Font Awesome 6** (фронтенд)
- **Lombok** (для сокращения кода)
- **Maven** (сборка)
